# Int3rceptor Official Website

This directory contains the source code for the official Int3rceptor landing page!

## Tech Stack

-   **Framework**: Vue 3
-   **Build Tool**: Vite
-   **Styling**: Vanilla CSS (Cyberpunk Theme)

## Development

1. Install dependencies:

    ```bash
    npm install
    ```

2. Start the development server:

    ```bash
    npm run dev
    ```

3. Open `http://localhost:5173` in your browser.

## Building for Production

To create a production build:

```bash
npm run build
```

The output will be in the `dist` directory, ready to be deployed to GitHub Pages, Netlify, or Vercel.

## Assets

Screenshots and images are located in `public/assets/screenshots/`. These are copied from the main project's `assets` directory.
